Annex NEP applies the Harmonic Orbital Mechanics framework to Neptune's satellite and ring system. Using the framework's established operators, including admissibility constraints, phase-lock relationships, lattice occupancy rules, and curvature-frequency mappings, the paper derives a structural model of Neptune's orbital architecture.

The analysis identifies harmonic cavities, lattice node assignments, ring boundaries, and phase relationships throughout the system. Derived values are compared against NASA/JPL ephemerides and observational measurements. The same operators are applied throughout the study without introducing system-specific fitting parameters.

This work serves as the Neptune-system application of the Harmonic Orbital Mechanics framework. Foundational derivations are cited from the underlying framework papers. The focus of this annex is the derivation, measurement comparison, and structural interpretation of the Neptune system using those established operators.
